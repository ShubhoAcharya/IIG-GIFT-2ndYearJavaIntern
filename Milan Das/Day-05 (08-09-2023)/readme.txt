    =   What is PATH environment variable?
        It is a variable name that is set by the Operating System.

    =   Why is PATH environment variable set?
        The PATH environment variable is set to enlist the name of the directories
        separated by a semicolon (;). The command that is fired from the command
        line are searched in the folders that are enlisted in the PATH environment
        variable.

    =   The java compiler compiles Java source programs (.java) to generate bycode

        back to the java source file. This process is called "Reflection".

    =   To reengineer the bycode file back to java source file, Java has provided
